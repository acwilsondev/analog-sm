# High Level Overview

The high-level vision of this project is Indie Myspace-like called **Analog SM** (Analog Social Media).

Target Environment: Self-hostable, docker-compose
Expected Userbase; 1-10 people, limited concurrency
Initial features out of the gate:

- Post to timeline
  - Short, long text
  - Photo, Photo set
- Timeline is *strictly* temporal order
- Can add people as friends to recieve their timeline updates
- Host a profile
- No custom CSS or html for now.
- ALL standard features for operating a production site.

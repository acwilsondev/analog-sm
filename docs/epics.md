# Epics Overview: Analog SM v0.1 - "The Digital Living Room"

This version focuses on the core features needed for a self-hostable, private social space for small groups.

## 1. Profile & Identity
Establish the basic identity of users within the instance.

## 2. Content Creation
Enable users to post text updates and photo sets.

## 3. Social Connectivity
Implement a simple friend request system to connect users.

## 4. The Timeline
Present posts in a simple, reverse-chronological order with zero algorithms.
